# Archive

Historical planning notes, verification reports, and research writeups live here.

- `reports/` contains dated implementation and verification reports.
- `plans/` contains implementation plans that are no longer the active source of truth.
- `research/` contains long-form background analysis that informs the project but is not part of the user-facing docs set.
- The active roadmap now lives at [`../technical-roadmap.md`](../technical-roadmap.md); superseded roadmap drafts should be moved into `plans/`.
